verilator入门笔记

verilator主页

安装

linux

archlinux:

sudo pacman -S verilator

debian/ubuntu

sudo apt-get install verilator

windows

使用wsl或MinGW/Cygwin编译安装

编译安装

常用参数解释

  • —cc 生成C库
  • —sc 生成systemc
  • —exe 包含测试程序创建makefile生成模拟可执行文件
  • —build 自动调用编译器构建最终的可执行文件
  • —binary 生成二进制文件
  • —trace 允许执行程序时生成波形文件
  • —trace-fst 生成fst格式的波形文件

项目创建

创建一个工程文件夹

mkdir vtest ; cd vtest

分别为verilog和c++测试程序创建文件夹

mkdir csrc vsrc

编写verilog程序

vim vsrc/our_OnOff.v

示例程序创建了our_OnOff模块，a、b作为输入信号，f取a、b异或作为输出

编写测试程序

vim csrc/main.c

测试程序生成随机的a、b作为输入并导出vcd图像

Makefile

all目标：使用verilator生成c库和makefile(Vour_OnOff.mk)，使用make编译

sim目标：在满足all目标后运行可执行文件并使用gtkwave查看波形

clean：删除obj_dir和wave.vcd
